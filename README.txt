============================================================
  議事録作成補助ツール v0.2.0  利用手順書
============================================================

【このツールでできること】
  - マイク録音 / 音声・動画ファイルからの文字起こし
  - 話者分離（誰が話したか識別）
  - AI による要約（Claude API 使用）
  - テキスト / PDF / Word / CSV 形式でのエクスポート
  - 置換機能（Ctrl+F）


============================================================
  はじめに（初回セットアップ）
============================================================

【STEP 1】インストーラーを実行する
  - 「議事録作成補助ツール Setup 0.2.0.exe」をダブルクリック
  - 画面の指示に従ってインストール
  - 完了するとスタートメニューにアプリが登録されます


【STEP 2】セットアップバッチを実行する
  - 「setup.bat」を右クリック →「管理者として実行」
  - 画面の指示に従って進める（5ステップ）
    1. Python 3.12 の確認・自動インストール
    2. faster-whisper のインストール（文字起こしエンジン）
    3. pyannote.audio のインストール（話者分離機能）
       ※ PyTorch を含むため 10〜20 分かかる場合があります
    4. FFmpeg の確認・自動インストール（音声・動画処理）
    5. API キーの設定
       - Anthropic API キー（AI要約機能）
       - HuggingFace トークン（話者分離機能）

  ※ Python・FFmpeg は未インストールでも自動でインストールされます
  ※ インターネット接続が必要です


【STEP 3】アプリを起動する
  - スタートメニューから「議事録作成補助ツール」を起動


============================================================
  各機能の使い方
============================================================

■ 録音して文字起こしする
  1. 「録音」ボタンをクリック → マイクへのアクセスを許可
  2. 会議や音声を録音する
  3. 「停止」ボタンをクリック（録音ファイルは自動保存）
  4. 「文字起こし」タブで「文字起こし」ボタンをクリック
     ※ 録音ファイルは以下の2か所に保存されます
       ・アプリの履歴フォルダ
       ・ダウンロードフォルダ


■ 音声・動画ファイルから文字起こしする
  1. 「音声/動画ファイル選択」ボタンをクリック
  2. ファイルを選択（WAV / MP3 / MP4 / MOV など対応）
  3. 「文字起こし」ボタンをクリック


■ 話者分離（誰が話したか識別する）
  1. 「話者分離」チェックボックスをオンにする
  2. 「文字起こし」ボタンをクリック
  ※ 初回実行時はモデルのダウンロードで時間がかかります
  ※ 別途 Python 3.12 と pyannote.audio が必要です（後述）


■ AI 要約する
  1. 文字起こしを完了させる
  2. 「要約」タブをクリック
  3. 「AI要約」ボタンをクリック
  ※ ANTHROPIC_API_KEY の設定が必要です（STEP 2 で設定）


■ エクスポートする
  1. 文字起こしを完了させる
  2. 「エクスポート ▾」ボタンをクリック
  3. 形式を選択（テキスト / PDF / Word / CSV）


■ テキストを置換する
  1. 文字起こし結果のテキストエリアをクリック
  2. Ctrl + F キーを押す
  3. 検索文字と置換文字を入力して「置換」または「すべて置換」


============================================================
  話者分離・AI要約を使うための準備（setup.bat 実行前に確認）
============================================================

■ AI要約（Claude API）
  - Anthropic のアカウントが必要です（有料）
  - API キーの取得先: https://console.anthropic.com/
  - setup.bat 実行時に入力を求められます

■ 話者分離（HuggingFace）
  - HuggingFace の無料アカウントが必要です
  1. https://huggingface.co/ で無料アカウントを作成
  2. 以下のページで利用規約に同意（要ログイン）
     https://huggingface.co/pyannote/speaker-diarization-3.1
  3. Settings → Access Tokens でトークンを取得
  - setup.bat 実行時に入力を求められます


============================================================
  API キーを後から設定・変更する方法
============================================================

  1. 以下のファイルをメモ帳で開く:
       C:\Users\（ユーザー名）\AppData\Local\Programs\議事録作成補助ツール\.env

  2. 以下の行を追加または編集して保存:
       ANTHROPIC_API_KEY=sk-ant-xxxxxxxxxxxxxxxxxx

  3. アプリを再起動する


============================================================
  トラブルシューティング
============================================================

Q. 文字起こしが失敗する
  → Python が正しくインストールされているか確認
  → faster-whisper が pip でインストールされているか確認
  → FFmpeg が PATH に登録されているか確認
     コマンドプロンプトで「ffmpeg -version」と入力して確認

Q. AI要約が失敗する
  → .env ファイルに ANTHROPIC_API_KEY が正しく設定されているか確認
  → API キーの先頭が「sk-ant-」で始まっているか確認

Q. アプリが起動しない
  → 一度アンインストールして再インストールを試みる
  → ログファイルを確認する:
       C:\Users\（ユーザー名）\AppData\Local\Programs\議事録作成補助ツール\log\
