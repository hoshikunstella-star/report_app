============================================================
  議事録作成補助ツール v0.2.0  利用手順書
============================================================

【このツールでできること】
  - マイク録音 / 音声・動画ファイルからの文字起こし
  - 話者分離（誰が話したか識別）※有料プランのみ
  - AI による要約（Claude API 使用）※有料プランのみ
  - テキスト / PDF / Word / CSV 形式でのエクスポート
  - 置換機能（Ctrl+F）
  - 自動アップデート（起動時に新バージョンを自動検知）

【プラン】
  - 無料プラン：文字起こし・エクスポート・履歴保存が利用可能
  - 有料プラン（月額制）：話者分離・AI要約が追加で利用可能
    ※ 有料プランへの切り替えはアプリ内の「有料プランに切り替え」ボタンから


============================================================
  はじめに（初回セットアップ）【Windows】
============================================================

【STEP 1】インストーラーを実行する
  - 「議事録作成補助ツール Setup 0.2.0.exe」をダブルクリック
  - 画面の指示に従ってインストール
  - 完了するとスタートメニューにアプリが登録されます


【STEP 2】セットアップバッチを実行する
  - 「setup.bat」を右クリック →「管理者として実行」
  - 画面の指示に従って進める（5ステップ）
    1. Python 3.12 の確認・自動インストール
    2. faster-whisper のインストール（文字起こしエンジン）
    3. pyannote.audio のインストール（話者分離機能）
       ※ PyTorch を含むため 10〜20 分かかる場合があります
    4. FFmpeg の確認・自動インストール（音声・動画処理）
    5. サーバー接続 URL（API_BASE_URL）の入力

  ※ Python・FFmpeg は未インストールでも自動でインストールされます
  ※ インターネット接続が必要です


【STEP 3】アプリを起動する
  - スタートメニューから「議事録作成補助ツール」を起動


============================================================
  はじめに（初回セットアップ）【macOS】
============================================================

【STEP 1】アプリをインストールする
  - 「議事録作成補助ツール-x.x.x.dmg」を開く
  - アプリをアプリケーションフォルダにドラッグ


【STEP 2】セットアップスクリプトを実行する
  - ターミナルを開き、setup.sh があるフォルダで以下を実行:
      chmod +x setup.sh && ./setup.sh
  ※ chmod +x は Mac で .sh ファイルを実行するための権限付与です（初回のみ）
  - 画面の指示に従って Python・ffmpeg・pyannote のインストールと
    サーバー接続 URL の設定を行います


【STEP 3】アプリを起動する（初回のセキュリティ警告について）
  macOS はコード署名がないアプリを警告します。以下の手順で開いてください:
  1. Finder でアプリを右クリック（または Control+クリック）
  2. 「開く」を選択
  3. 警告ダイアログで「開く」をクリック
  ※ 2回目以降は通常通り起動できます
  ※ 自動アップデート後の新バージョンでも同様の操作が必要な場合があります


============================================================
  各機能の使い方
============================================================

■ 録音して文字起こしする
  1. マイクセレクトボックスで使用するマイクを選択
  2. 「録音」ボタンをクリック → マイクへのアクセスを許可
  3. 会議や音声を録音する
  4. 「停止」ボタンをクリック（録音ファイルは自動保存）
  5. 「文字起こし」ボタンをクリック
     ※ 録音ファイルは以下の2か所に保存されます
       ・アプリの履歴フォルダ
       ・ダウンロードフォルダ


■ 音声・動画ファイルから文字起こしする
  1. 「音声/動画ファイル選択」ボタンをクリック
  2. ファイルを選択（WAV / MP3 / MP4 / MOV など対応）
  3. 「文字起こし」ボタンをクリック


■ 話者分離（誰が話したか識別する）※有料プランのみ
  1. 「話者分離」チェックボックスをオンにする
  2. 「文字起こし」ボタンをクリック
  ※ 初回実行時はモデルのダウンロードで時間がかかります
  ※ 話者は「Speaker_A」「Speaker_B」のように表示されます


■ AI 要約する ※有料プランのみ
  1. 文字起こしを完了させる
  2. 「要約」タブをクリック
  3. 「AI要約」ボタンをクリック
  ※ 文字起こしの言語を自動検出し、同じ言語で要約します


■ エクスポートする
  1. 文字起こしを完了させる
  2. 「エクスポート ▾」ボタンをクリック
  3. 形式を選択（テキスト / PDF / Word / CSV）


■ テキストを置換する
  1. 文字起こし結果のテキストエリアをクリック
  2. Ctrl + F キーを押す（Mac は Cmd + F）
  3. 検索文字と置換文字を入力して「置換」または「すべて置換」


============================================================
  接続設定を後から変更する方法
============================================================

【Windows】
  以下のファイルをメモ帳で開いて編集:
    C:\Users\（ユーザー名）\AppData\Local\Programs\議事録作成補助ツール\.env

【macOS】
  以下のファイルをテキストエディタで開いて編集:
    ~/Library/Application Support/議事録作成補助ツール/.env

  設定内容:
    API_BASE_URL=https://（Railway サーバーの URL）

  編集後はアプリを再起動してください。


============================================================
  トラブルシューティング
============================================================

Q. 文字起こしが失敗する
  → Python が正しくインストールされているか確認
  → faster-whisper が pip でインストールされているか確認
  → FFmpeg が PATH に登録されているか確認
     コマンドプロンプトで「ffmpeg -version」と入力して確認

Q. AI要約が失敗する
  → ログインしているか確認（有料プランが必要）
  → インターネット接続を確認

Q. 話者分離が失敗する
  → pyannote.audio が pip でインストールされているか確認
  → インターネット接続を確認（初回はモデルのダウンロードが必要）

Q. アプリが起動しない
  → 一度アンインストールして再インストールを試みる
  → ログファイルを確認する:
     【Windows】C:\Users\（ユーザー名）\AppData\Roaming\議事録作成補助ツール\log\
     【macOS】  ~/Library/Application Support/議事録作成補助ツール/log/

Q. macOS でアプリが開けない（セキュリティ警告）
  → アプリを右クリック →「開く」→「開く」で起動
