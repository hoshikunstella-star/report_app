# 議事録作成補助ツール

音声・動画ファイルから文字起こし・AI要約・議事録作成を行うデスクトップアプリです。

---

## 動作環境

| 項目 | 内容 |
|---|---|
| OS | Windows 10/11、macOS 12以降 |
| Python | 3.10 以上（文字起こし・話者分離に使用） |
| ffmpeg | MP4/MOV など動画ファイルの処理に必要 |

---

## インストール手順

### Windows

1. `議事録作成補助ツール Setup x.x.x.exe`（インストーラー版）または `議事録作成補助ツール x.x.x.exe`（ポータブル版）を実行
2. `setup.bat` をダブルクリックしてセットアップを実行
   - Python・ffmpeg・pyannote.audio のインストール
   - サーバー接続 URL（`API_BASE_URL`）の設定

### macOS

1. `議事録作成補助ツール-x.x.x.dmg` を開き、アプリをアプリケーションフォルダにドラッグ
2. ターミナルを開き、`setup.sh` があるフォルダで以下を実行

```bash
chmod +x setup.sh && ./setup.sh
```

> `chmod +x` は Mac で `.sh` ファイルを実行するために必要な権限付与です。初回のみ必要です。

画面の指示に従って以下を設定します：
- Python・ffmpeg・pyannote.audio のインストール
- サーバー接続 URL（`API_BASE_URL`）の設定
- 設定ファイルは `~/Library/Application Support/議事録作成補助ツール/.env` に自動保存

#### ⚠️ macOS セキュリティ警告について

このアプリは **Apple Developer 証明書（コード署名）を使用していません**。
そのため、初回起動時に以下のような警告が表示されます。

> "議事録作成補助ツール" は、Appleによるチェックを受けていません。

**回避手順：**

1. Finder でアプリを**右クリック**（または Control+クリック）
2. 「開く」を選択
3. 警告ダイアログで「開く」をクリック

2回目以降は通常通り起動できます。

> 自動アップデート後の新バージョンでも同様の操作が必要な場合があります。

## 自動アップデート

アプリ起動時に新バージョンを自動検知します。

- **Windows（インストーラー版）**：新バージョン検知時にダウンロード確認ダイアログを表示し、再起動時に適用
- **Windows（ポータブル版）**：自動アップデート対象外。新しい exe を手動で差し替えてください
- **macOS**：新バージョン検知時にダウンロード確認ダイアログを表示し、再起動して適用。新バージョン初回起動時にセキュリティ警告が出る場合があります（上記参照）
